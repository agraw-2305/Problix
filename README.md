# Problem to Product AI

Directory scaffold for an app that turns problem statements into product ideas and a roadmap.

Structure:

- backend/
  - main.py: Entry point for the backend service (API placeholder)
  - ai/
    - problem_parser.py: Parse raw problem input into structure
    - idea_generator.py: Generate ideas from the structured problem
    - roadmap_builder.py: Build a roadmap from a chosen idea
  - schemas/: Python package for future Pydantic/dataclass schemas

- frontend/
  - pages/: Frontend pages (framework-agnostic placeholder)
  - components/: Reusable UI components placeholder

- prompts/
  - problem_analysis.txt: Prompt to analyze problem statements
  - product_ideation.txt: Prompt to generate product concepts

## Getting Started

- Python 3.10+
- Node (optional for frontend)
